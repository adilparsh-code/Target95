# Target95 Project Rules

## Project

Target95 is an AI-powered learning platform for ICSE & ISC Computer Science students.

## Tech Stack

- Next.js 15 (App Router)
- React
- Tailwind CSS
- Firebase (Future)

## Coding Rules

- Never rewrite working code.
- Keep components reusable.
- One component = One responsibility.
- Use Tailwind CSS only.
- Prefer functional components.
- Mobile-first responsive design.
- Keep architecture scalable.
- Avoid unnecessary dependencies.
- Explain architectural changes before implementing.
- Generate production-ready code.

## Folder Structure

src/
├── app/
├── components/
├── data/
├── hooks/
├── lib/

## Git Rules

After every completed feature:

1. Test the feature.
2. Ensure the app builds successfully.
3. Commit with a meaningful message.
4. Push to GitHub.

## Current Focus

- Improve Question Filters
- Enhance Search
- Maintain clean architecture
- Do not break existing functionality
